BOURASS AMEGA DU II PROJET PYTHON 

_____________________________________________________________
Guide d'utilisation dans le terminale :

python3 cli.py
python3 cli.py -h
python3 cli.py -d .
python3 cli.py -f music.mp3 (mettre le chemin du son)
python3 cli.py -p music.mp3 (mettre le chemin du son)
python3 cli.py -d ./music/ -o playlist.xspf
python3 gui.py
______________________________________________________________

Installation pour faire marcher le programme :

Vérifiez si vous avez bien python et pip
python3 -m ensurepip (pour vérifier si pip est bien installé) 
pip install mutagen Pillow pygame (installer les bibliothèques)
